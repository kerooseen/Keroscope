# Keroscope

Keroscope is a Python graphical application (Tkinter) designed to analyze and remove metadata from your files to protect your privacy.

## Features
**Analysis:** Extract and view hidden metadata.

**Cleaning:** Permanently delete metadata from your files.

**Simplicity:** Quick file import via the graphical file explorer.
